# Personal Profile Generator

4th Lab Activity for IT 9a/L (Course Code: 4561).

_Disclaimer: This project was created for educational and institutional purposes._

This project is a simple web application that allows users to generate a personal profile page by filling out a form with their details, including profile and banner pictures.

## Features

- Form to input personal details (Name, Birthday, Course, Email, Gender, Hobbies, Biography).
- Upload profile and banner images.
- Generated profile view mimicking a social media page ("Pisbok").
